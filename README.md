# gueden.com

Personal site of Alejandro Guerra. Fullstack developer and solution architect based in Madrid.

## Stack

- Next.js 16 (App Router, Turbopack)
- React 19
- Tailwind CSS 4
- Contentlayer2 (MDX)
- next-intl (ES/EN)
- Vercel

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
